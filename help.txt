Instalar Nix:
    1. sh <(curl --proto '=https' --tlsv1.2 -L https://nixos.org/nix/install) --daemon
    2. curl --proto '=https' --tlsv1.2 -sSf -L https://install.determinate.systems/nix | sh -s -- install (RECOMENDADO)

Instalar direnv (não obrigatório):
    Fedora: sudo dnf install direnv
    Ubuntu: sudo apt install direnv

Comandos:
    shell.nix -> nix-shell
    flake.nix -> nix develop

Comandos com zsh:
    nix-shell --run zsh
    nix develop -c zsh

Verificar se esta no shell do nix:
echo $IN_NIX_SHELL -> impure
